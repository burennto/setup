# Homebrew

## Install
```
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Brew Bundle
```
$ brew tap Homebrew/bundle
$ brew bundle

# to view any post-install caveats
$ brew info git
```

## Brew Services
```
  Installed via Brewfile

  $ brew services start postgresql
```
